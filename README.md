# Advent of code 2025

[Challenge](https://adventofcode.com/2025)
## New!

Now using [advent-of-code-data](https://pypi.org/project/advent-of-code-data/). Remember to export your session ID, explained on their docs.

Run every puzzle day!

```bash
uv run aocd > d<day>/r.in
```


Or run for a specific day: (for example 2025 day 5)

```bash
uv run aocd 5 2025 > d5/r.in
```
